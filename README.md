# vue-vk

> Provides a simple layer on top of the VK widget iframe api.


## Usage:

* install via `npm i vue-vk`
* register the components with

```javascript
import VKWidgets from 'vue-vk';
import Vue from 'vue';

Vue.use(VKWidgets);
```
